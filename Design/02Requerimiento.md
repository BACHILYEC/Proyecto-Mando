# Requerimientos
1. El usuario puede escoger una de las siguiente opciones: "Iniciar juego", "visualizar marcador", "vaciar marcador" o "salir del juego".  
2. El usuario debe elegir uno de los diferentes personajes disponibles de la trivia para jugar.
3. El usuario escoge una de las diferentes opciones de categorías para la trivia y visualiza una animación de descarga de las mismas.
4. Si el usuario escoge la respuesta correcta, la puntuación aumenta en un punto.
5. Si el usuario escoge la respuesta equivocada tiene dos intentos más para acertar.
6. El sistema debe validar si la respuesta elegida por el usuario es correcta o no.
7. El usuario gana el juego si contesta correctamente el banco de preguntas de cada categoría y se muestra la puntuación.
8. El usuario pierde cuando se agotan los 3 intentos proporcionados y se muestra la puntuación.
9. El usuario gana la categoría al contestar correctamente las preguntas de la misma y puede volver a jugar eligiendo otra categoría o visualizar la tabla de puntuación.
10. El usuario puede visualizar su puntuación final al ganar o perder el juego.
