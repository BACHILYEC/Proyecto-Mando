# Requerimientos
1. El usuario puede escoger una de las siguiente opciones: "Iniciar juego", "visualizar marcador", "vaciar marcador" o "salir del juego".  
2. Para jugar, el usuario debe escoger un personaje entre "Neil Armstrong", "Madame Curie", "Nikola Tesla", "Pat_Mic" o "Peter Pan".
3. El usuario escoge una categoría de preguntas entre Disney, Cultura General, Tipos de Datos en Programación, planetas y electrónica, también visualiza una animación de descarga de las mismas.
4. Si el usuario escoge la respuesta correcta, la puntuación aumenta en un punto.
5. Si el usuario escoge la respuesta equivocada tiene dos intentos más para acertar.
6. El sistema debe validar si la respuesta elegidad por el usuario es correcta o no.
6. El usuario gana el juego si contesta correctamente el banco de preguntas de cada categoría.

7. El usuario gana la categoría al contestar correctamente las preguntas de la misma y puede volver a jugar eligiendo otra categoría.
8. El usuario puede visualizar su puntuación final al finalizar el juego.
